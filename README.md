# cURLy

snake game with cURL

```bash
# for Linux/Mac
curl -fsSL cURLy.gjt.io/install.sh | bash

# TODO : for Windows 
```

## pre-requisites

- [Go](https://go.dev/doc/install)
- [Air](https://github.com/air-verse/air?tab=readme-ov-file#installation)

## tasks

```bash
# run cURLy tui
go run ./cmd/tui

# run web server
PORT=3000 go run ./cmd/server/
```
